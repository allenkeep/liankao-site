## 深圳北理莫斯科大学2026年硕士招生

深圳北理莫斯科大学发布2026年硕士研究生招生简章，提供14个专业方向，自主招生录取。

### 学校简介

深圳北理莫斯科大学是经教育部批准，由深圳市人民政府、北京理工大学和莫斯科国立罗蒙诺索夫大学三方合作设立的具有独立法人资格的中外合作大学，实施与莫斯科大学联合培养硕士学位教育。

### 招生专业及人数

| 专业方向 | 招生人数 |
|---------|---------|
| 对外俄语教学 | 45人 |
| 人工智能方法与算法 | 15人 |
| 应用数学与技术 | 45人 |
| 国际经济与商业 | 25人 |
| 国际经济(商)法 | 20人 |
| 国家和企业人力资源发展管理 | 40人 |
| 战略发展和区域管理 | 40人 |
| 纳米生物技术 | 15人 |
| 全球环境变化生态学与管理 | 15人 |
| 海洋生物多样性与生物资源 | 12人 |
| 城市生态学与区域规划 | 20人 |
| 基础材料学 | 15人 |
| 光电信息科学 | 15人 |
| 放射化学与放射生态学 | 15人 |

### 项目信息

- **学制**：2年
- **学费**：72000元/学年
- **培养模式**：自主招生，不参加全国统考
- **学位授予**：符合条件者颁发莫斯科大学学位证书，可在教育部留学服务中心进行学历学位认证

### 报考建议

深圳北理莫斯科大学的优势在于毕业可获得莫斯科大学（QS排名前100）学位证书，且无需出国。人工智能、国际经济与商业、人力资源管理等热门专业竞争较激烈，建议尽早提交申请。
