## 北航经管学院启动2027年非全日制MEM知行成长营

北京航空航天大学经济管理学院于6月2日发布通知，宣布启动2027年非全日制工程管理硕士（MEM）知行成长营活动。活动时间为2026年6月至10月，面向广大考生开放报名。

**项目基本信息：**

| 项目 | 信息 |
|------|------|
| 专业代码 | 125601 工程管理 |
| 学习方式 | 非全日制 |
| 培养方向 | 项目管理、数据与信息工程管理、智慧工程管理 |
| 培养费用 | 共计16.6万元（学制内8.3万元/生·学年） |
| 培养学制 | 二年 |
| 班型设置 | 周末班（周六、周日上课） |
| 就业类型 | 定向就业 |

**项目特色：**
- 北航于2010年成为我国首批获得工程管理硕士专业学位授权点的院校
- 依托管理科学与工程高峰学科底蕴
- 面向航空航天、智能制造、新能源、新基建、具身智能工程等关键领域
- 构建"思政课+理论课+专业技术课+综合素养课"四位一体课程体系
- 2026年录取非全日制MEM学生283人
