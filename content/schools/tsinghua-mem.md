# 清华大学MEM项目介绍

清华大学作为国内顶尖学府，其工程管理硕士（MEM）项目备受瞩目。

## 项目概况

| 项目 | 信息 |
|------|------|
| 培养学院 | 工业工程系 |
| 招生人数 | 约200人 |
| 学制 | 2-3年 |
| 学费 | 16.8万元 |
| 授课方式 | 周末班 |

## 课程体系

### 核心课程
- 工程管理导论
- 工程经济学
- 项目管理
- 供应链管理
- 质量管理

### 特色方向
1. 信息化管理
2. 先进制造
3. 能源与环境
4. 工程建设

## 报考条件

1. 中华人民共和国公民
2. 拥护中国共产党的领导
3. 身体健康状况符合规定
4. 学历要求：
   - 本科毕业后3年以上工作经验
   - 专科毕业后5年以上工作经验
   - 硕士/博士毕业后2年以上工作经验

## 考试科目

| 科目 | 分值 |
|------|------|
| 管理类联考综合 | 200分 |
| 英语二 | 100分 |

<!--more-->

## 提前面试

清华MEM采用**提前面试**政策，面试时间通常在每年的7-10月。

### 面试内容
- 个人综合素质面试
- 英语口语测试
- 政治理论测试

### 面试技巧
1. 突出技术背景优势
2. 展示项目管理经验
3. 表达清晰的职业规划

## 校友资源

清华MEM项目拥有丰富的校友资源，涵盖：
- 大型国企高管
- 知名外企经理人
- 科技创新创业者
- 政府部门管理者

## 就业前景

毕业 生就业方向广泛，主要集中在：
- 互联网科技企业
- 制造业
- 金融机构
- 咨询公司
- 政府部门
