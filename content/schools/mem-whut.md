# 武汉理工大学工程管理硕士（MEM）

## 基本信息

| 项目 | 信息 |
|------|------|
| **院校名称** | 武汉理工大学 |
| **项目名称** | 工程管理硕士（MEM） |
| **培养学院** | 管理学院/土木工程与建筑学院 |
| **项目类型** | 非全日制 |
| **学制** | 2-3年 |
| **学费** | 以学校最终公布为准 |
| **上课方式** | 周末授课 |

| **官方网址** | https://自查 |

## 项目特色

武汉理工大学MEM项目依托学校在材料、交通、汽车等领域的优势：

- **建材建工特色**：依托学校在建材建工领域的传统优势
- **汽车交通优势**：覆盖汽车、交通等领域工程管理
- **行业深度融合**：与大型企业深度合作培养人才
- **复合型培养**：工程技术与管理知识融合

## 研究方向

| 方向 | 简介 |
|------|------|
| 建设工程管理 | 建筑工程项目管理 |
| 交通工程管理 | 交通基础设施管理 |
| 物流工程管理 | 物流与供应链管理 |

## 报考条件

- 大学本科毕业后有3年以上工作经验
- 高职高专毕业或本科结业后有5年以上工作经验
- 获硕士、博士学位后有2年以上工作经验

**注**：仅招收在职定向就业人员

## 考试科目

- 初试：199管理类综合能力 + 204英语二
- 复试：专业课笔试、思想政治理论，综合面试

## 证书授予

完成规定学分、通过学位论文答辩后，授予：
- 武汉理工大学硕士研究生毕业证书
- 工程管理硕士学位证书

**来源**：以学校官方公布信息为准
