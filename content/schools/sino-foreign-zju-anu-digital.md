## 项目简介
浙江大学与澳大利亚国立大学合作举办数字化转型与创业硕士学位教育项目，经教育部批准设立（批准书编号：MOE33AU1A202526005N）。项目免全国联考，采用申请制入学，毕业获外方硕士学位，可在中国教育部留学服务中心进行学历学位认证。

## 审批信息

| 项目信息 | 详情 |
|---------|------|
| 中方办学者 | 浙江大学 |
| 外方办学者 | 澳大利亚国立大学 |
| 开设专业 | 数字化转型与创业硕士学位教育项目 |
| 批准书编号 | MOE33AU1A202526005N |
| 学制 | 2年 |
| 每期招生人数 | 50人 |
| 招生起止年份 | 2025年—2030年 |
| 招生方式 | 自主招生 |
| 学费 | 以官方公布为准 |
| 学历认证 | 可留服认证 |

## 申请条件
- 本科及以上学历
- 相关专业背景优先
- 具备一定英语能力
- 具体条件以当年招生简章为准

## 项目特色
- 教育部批准，学历可留服认证
- 免全国联考，申请制入学
- 中外双方师资联合授课
- 国际化课程体系
