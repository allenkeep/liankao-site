# 对外经济贸易大学-法国诺欧商学院零售管理硕士

## 项目基本信息

| 项目名称 | 内容 |
|---------|------|
| **中方院校** | 对外经济贸易大学 |
| **外方院校** | 法国诺欧商学院 (NEOMA Business School) |
| **专业方向** | 零售管理 (MSc in Retail Management) |
| **学制** | 2年 |
| **总学费** | 20.96万元 |
| **是否留服认证** | 是 |
| **上课方式** | 周末集中授课 |
| **招生人数** | 55人/年 |
| **入学方式** | 自主招生（笔试+面试） |
| **上课地点** | 对外经济贸易大学校内（北京） |
| **批准书编号** | MOE11FR1A20060140N |

## 项目特色

- **全国首家零售管理硕士项目**：弥补国内零售管理领域专业教育空白
- **国际领先课程**：问题导向型教学模式，引进国际先进教育理念
- **双校师资**：中法两国著名教授授课，80%法方教师具有企业高管经验
- **无需出国**：全程在中国境内完成教学
- **高质量认证**：三大商学院认证(EQUIS/AACSB/AMBA)

## 招生条件

1. 具有中国教育部认可的本科及以上学历
2. 零售行业或管理岗位在职人员，具有3年及以上工作经验
3. 具有一定英语水平
4. 通过项目笔试及面试考核

## 课程设置

### 中方课程（10门，20学分）
- 新时代中国特色社会主义理论与实践
- 供应链管理
- 组织行为与管理
- 市场调研
- 品牌营销与管理
- 谈判技巧
- 项目投资与管理
- 国际投资与合作
- 数字营销
- 特许经营

### 法方课程（14门，40学分）
- 全球零售业概况
- 零售战略
- 零售领导力
- 零售金融
- 零售市场营销
- 国际零售
- 企业社会责任
- 空间技术管理
- 消费者行为学
- 客户关系管理
- 产品创新
- 成本核算与绩效管理
- 研究方法论
- 商业计划

## 毕业获证

- 法国诺欧商学院零售管理理学硕士学位证书（全球认可）
- 对外经济贸易大学零售管理硕士项目结业证书
- 可申请教育部留学服务中心学历学位认证
