# 南京大学工程管理硕士（MEM）

## 基本信息

| 项目 | 信息 |
|------|------|
| **院校名称** | 南京大学 |
| **项目名称** | 工程管理硕士（MEM） |
| **培养学院** | 工程管理学院 |
| **项目类型** | 非全日制 |
| **学制** | 3年（弹性学制，最长不超过5年） |
| **学费** | 19.8万元（全程） |
| **上课方式** | 周六至周日分段集中授课 |
| **授课地点** | 南京大学鼓楼校区、苏州校区 |
| **招生人数** | 计划142人左右 |
| **复试线** | 参考南京大学自主划线 |
| **院校属性** | 985、211、双一流A类 |
| **官方网址** | https://www.nju.edu.cn |

## 培养方向

- 项目管理与数智化技术
- 智慧物流与供应链管理
- 智能工业工程与管理
- 金融科技与金融工程

## 项目特色

- **"1+1+1"产教融合师资**：由校内一流教师、校外实践专家及国内外专业人士组成教学团队
- **智能系统管理课程体系**：构建"基础理论+智能工具+应用场景"的复合课程体系，全国首开面向MEM定制化的人工智能原理与技术基础课程
- **双城办学与弹性学制**：实行南京与苏州两地办学，满足不同区域学生需求
- **国际化认证衔接**：与IPMP国际项目经理资质认证合作，学生可申请IPMP D级认证
- **江苏省MEM高校联盟牵头单位**：资源共享，平台优势明显

## 课程设置

### 核心课程
- 工程管理基础理论
- 系统集成方法论
- 人工智能原理与技术基础
- 工程管理定量化技术
- 工程经济与金融管理
- 工程现代社会治理
- 工程系统综合品质管理

### 方向选修课
根据四大培养方向，开设项目管理、物流与供应链、工业工程、金融工程等领域的菜单式课程

### 实践教学
- 案例教学
- 沉浸式实践教学
- AI教学手段
- 企业诊断
- 项目咨询
- 工程管理案例竞赛

## 报考条件

1. 大学本科毕业后有3年及以上工作经验
2. 获得国家承认的高职高专毕业学历或大学本科结业后，有5年及以上工作经验
3. 获得硕士、博士学位后有2年及以上工作经验

*工作年限计算截止至2027年9月1日*

## 初试科目

| 科目 | 满分 |
|------|------|
| 管理类联考综合能力（199） | 200分 |
| 英语二（204） | 100分 |

## 学费缴纳

学费按学年分三次缴纳，每学年6.6万元

## 证书授予

完成培养方案规定学分、通过学位论文答辩，经南京大学学位评定委员会审核批准，可获得：
- 南京大学硕士研究生毕业证书（非全日制）
- 工程管理硕士（MEM）学位证书
