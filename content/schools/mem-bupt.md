# 北京邮电大学工程管理硕士（MEM）

| 项目 | 信息 |
|------|------|
| **院校名称** | 北京邮电大学 |
| **项目名称** | 工程管理硕士（MEM） |

| **项目类型** | 非全日制 |

## 项目特色

北京邮电大学MEM项目依托学校信息通信领域的独特优势：

- **首批MEM培养单位**：国务院学位委员会第一批授权的工程管理硕士培养单位
- **ICT+管理融合**：将信息技术与工程管理深度结合
- **三大研究方向**：项目管理、ICT项目和大数据工程
- **互联网+思维**：培养具有新思维的工程技术管理复合型人才
- **数智化特色**：利用新兴技术解决复杂工程问题

## 研究方向

| 方向 | 简介 |
|------|------|
| 项目管理 | 工程项目全生命周期管理 |
| ICT项目 | 信息与通信技术项目管理 |
| 大数据工程 | 大数据项目管理与工程实践 |

## 报考条件

- 大学本科毕业后有3年以上工作经验
- 高职高专毕业或本科结业后有5年以上工作经验
- 获硕士、博士学位后有2年以上工作经验

**注**：仅招收在职定向就业人员

## 考试科目

- 初试：199管理类综合能力 + 204英语二
- 复试：综合素质面试，英语口语，政治笔试

## 证书授予

完成规定学分、通过学位论文答辩后，授予：
- 北京邮电大学硕士研究生毕业证书
- 工程管理硕士学位证书
