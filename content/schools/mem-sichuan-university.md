# 四川大学工程管理硕士（MEM）

## 基本信息

| 项目 | 信息 |
|------|------|
| **院校名称** | 四川大学 |
| **项目名称** | 工程管理硕士（MEM） |
| **培养学院** | 商学院 |
| **项目类型** | 非全日制 |
| **学制** | 2.5年（最长不超过3年） |
| **学费** | 4.8万元（全程） |
| **上课方式** | 周末授课+集中授课 |
| **授课地点** | 四川大学望江校区（成都） |
| **招生人数** | MEM为重点专业之一，招生规模大 |
| **复试线** | 参考四川大学自主划线 |
| **院校属性** | 985、211、双一流A类 |
| **官方网址** | https://www.scu.edu.cn |

## 项目方向

- 工程项目管理
- 工业工程与管理
- 物流与供应链管理
- 建设工程管理
- 研发与创新管理

## 项目特色

- **西南地区历史最悠久**：始建于1896年，文化底蕴深厚
- **全国首批MEM培养单位**：2010年首批获准招生
- **双导师制培养**：校内导师+企业导师，强化实践能力
- **国际认证合作**：与IPMP国际项目管理专业资质认证合作，学员可免笔试申请IPMP认证
- **学科交叉优势**：整合工程、管理、信息技术等多学科资源
- **性价比高**：学费相对较低，是西南地区热门选择

## 课程设置

### 核心课程
- 项目管理学
- 系统工程
- 工程经济学
- 运营管理
- 风险管理
- 管理信息系统

### 方向选修课
- 工程项目管理
- 物流与供应链管理
- 信息技术与数字化转型
- 绿色工程与可持续发展

### 实践环节
- 企业调研
- 工程管理案例研讨
- 毕业论文（需通过答辩）

## 报考条件

1. 大学本科毕业后有3年以上工作经验
2. 获得国家承认的高职高专科毕业学历或大学本科结业后，有5年以上工作经验
3. 获得硕士、博士学位后有2年以上工作经验

*工作年限计算截止至2027年9月1日*

## 初试科目

| 科目 | 满分 |
|------|------|
| 管理类联考综合能力（199） | 200分 |
| 英语二（204） | 100分 |

## 复试内容

- 专业课笔试（项目管理学）
- 政治理论考试
- 综合面试（含英语能力测试）
- 初试成绩占比60%，复试成绩占比40%

## 证书授予

完成培养方案规定学分、通过学位论文答辩，经四川大学学位评定委员会审核批准，可获得：
- 四川大学硕士研究生毕业证书（非全日制）
- 工程管理硕士（MEM）学位证书（双证）
