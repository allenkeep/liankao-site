## 项目简介
中央财经大学与香港理工大学合作举办数智金融，经教育部批准设立（批准书编号：MOE11HK1A20262765N）。项目免全国联考，采用申请制入学，毕业获外方硕士学位，可在中国教育部留学服务中心进行学历学位认证。

## 审批信息

| 项目信息 | 详情 |
|---------|------|
| 中方办学者 | 中央财经大学 |
| 外方办学者 | 香港理工大学 |
| 开设专业 | 数智金融 |
| 批准书编号 | MOE11HK1A20262765N |
| 学制 | 3年 |
| 每期招生人数 | 40人 |
| 招生起止年份 | 2026年—2029年 |
| 招生方式 | 自主招生 |
| 学费 | 30万元 |
| 学历认证 | 可留服认证 |

## 申请条件
- 本科及以上学历
- 相关专业背景优先
- 具备一定英语能力
- 具体条件以当年招生简章为准

## 项目特色
- 教育部批准，学历可留服认证
- 免全国联考，申请制入学
- 中外双方师资联合授课
- 国际化课程体系
