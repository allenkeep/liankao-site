## 项目简介

江西财经大学与美国纽约理工学院(NYIT)合作的MBA项目，是教育部批准的中外合作办学项目（批准书编号：MOE36US1A20030485O）。2年学制，毕业可留服认证。

## 项目审批信息

| 项目 | 详情 |
|------|------|
| 项目名称 | 江西财经大学与美国纽约理工学院合作举办工商管理硕士学位教育项目 |
| 办学地址 | 江西省南昌市庐山南大道江西财经大学蛟桥园 |
| 中方办学者 | 江西财经大学 |
| 外方办学者 | New York Institute of Technology, USA（美国纽约理工学院） |
| 办学层次 | 外国硕士学位教育 |
| 学制 | 2年 |
| 每期招生人数 | 300人 |
| 招生起止年份 | 2003年—2027年（每年1期） |
| 招生方式 | 自主招生（招生标准不低于美国纽约理工学院在美国的标准） |
| 开设专业 | 工商管理（MBA） |
| 颁发证书 | 中方：无；外方：Master of Business Administration（工商管理硕士学位证书） |
| 审批机关 | 教育部 |
| 批准书编号 | MOE36US1A20030485O |
| 批准书有效期 | 2029年12月31日 |
