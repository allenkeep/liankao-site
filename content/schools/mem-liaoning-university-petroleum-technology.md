# 辽宁石油化工大学MEM项目介绍

## 学校简介

辽宁石油化工大学位于辽宁省抚顺市，是一所以石油石化为特色的多科性大学。学校工程管理专业办学历史悠久，在石油石化行业具有重要影响力。

## MEM项目简介

辽宁石油化工大学MEM项目依托学校石油石化学科优势，面向石油炼制、石油化工、油气储运等行业培养高级工程管理人才。

### 招生信息

| 项目 | 内容 |
|------|------|
| 招生院系 | 经济管理学院 |
| 培养方向 | 工程项目管理、石化工程管理、供应链管理 |
| 授课方式 | 周末授课 |
| 招生人数 | 约30人 |

### 学制学费

| 项目 | 内容 |
|------|------|
| 学制 | 3年 |
| 学费 | 约3.6万元（全程） |
| 授课安排 | 周末+部分集中 |

## 师资力量

学校拥有一支实践经验丰富的工程管理教师团队，多数教师具有石油石化企业工作经历或项目咨询经验。

## 就业前景

毕业生主要就业于石油石化企业、工程建设公司、设计院等，就业率保持较高水平。
