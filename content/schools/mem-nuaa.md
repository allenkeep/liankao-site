# 南京航空航天大学工程管理硕士（MEM）

## 基本信息

| 项目 | 信息 |
|------|------|
| **院校名称** | 南京航空航天大学 |
| **项目名称** | 工程管理硕士（MEM） |
| **培养学院** | 经济与管理学院（MBA教育中心） |
| **项目类型** | 非全日制 |
| **学制** | 2.5年（最长学习年限4.5年） |
| **学费** | 15.8万元（全程） |
| **上课方式** | 周末班/集中班/平时班 |
| **授课地点** | 南京航空航天大学将军路校区（南京） |
| **招生人数** | 约100人 |
| **院校属性** | 211，双一流 |
| **官方网址** | http://mba.nuaa.edu.cn |

## 项目特色

南京航空航天大学MEM项目依托学校在三航领域的独特优势：

- **PMI-GAC国际认证**：2022年通过全球项目管理教育认证中心认证
- **"三航"特色**：深度融合航空、航天、民航领域需求
- **管理科学与工程A-学科**：教育部学科评估获评A-等级
- **双导师制**：校内外导师联合指导
- **启航计划**：提供预审与交流机制

## 研究方向

| 方向 | 简介 |
|------|------|
| 复杂工程与项目管理 | 大型工程项目全生命周期管理 |
| 智能制造与服务工程管理 | 智能制造与服务业管理优化 |
| 大数据与工程信息管理 | 数据驱动的工程管理决策 |

## 报考条件

- 大学本科毕业后有2年以上工作经验
- 高职高专毕业或本科结业后有4年以上工作经验
- 获硕士、博士学位后有1年以上工作经验

**注**：仅招收在职定向就业人员

## 考试科目

- 初试：199管理类综合能力 + 204英语二
- 复试：思想政治理论，综合面试

## 证书授予

完成规定学分、通过学位论文答辩后，授予：
- 南京航空航天大学硕士研究生毕业证书
- 工程管理硕士学位证书
