# 东华大学工程管理硕士（MEM）

## 基本信息

| 项目 | 信息 |
|------|------|
| **院校名称** | 东华大学 |
| **项目名称** | 工程管理硕士（MEM） |
| **培养学院** | 旭日工商管理学院 |
| **项目类型** | 非全日制 |
| **学制** | 2.5年（最长学习年限5年） |
| **学费** | 16.8万元（全程） |
| **上课方式** | 周末授课或集中授课 |
| **授课地点** | 东华大学延安西路校区（上海） |
| **招生人数** | 约150人 |
| **院校属性** | 211，双一流 |
| **官方网址** | https://mba.dhu.edu.cn |

## 项目特色

东华大学MEM项目是国内第一批MEM培养院校：

- **首批MEM培养单位**：2010年成为我国第一批工程管理硕士培养院校
- **AMBA和BGA认证**：2021年通过AMBA和BGA（金牌）联合认证
- **工管结合**：整合纺织、材料、机械、计算机等优势学科资源
- **双导师制**：校内导师+企业导师共同指导
- **国际资质认证**：可申请IPMP国际项目经理资质认证

## 研究方向

| 方向 | 简介 |
|------|------|
| 项目工程管理 | 项目投融资、采购与合同管理、工程造价管理 |
| 人工智能与大数据管理 | 大数据分析与挖掘、人工智能原理与应用 |
| 机械工程与智能制造管理 | 智能制造系统、物流与供应链管理 |

## 报考条件

- 大学本科毕业后有3年以上工作经验
- 高职高专毕业或本科结业后有5年以上工作经验
- 获硕士、博士学位后有2年以上工作经验

**注**：仅招收在职定向就业人员

## 考试科目

- 初试：199管理类综合能力 + 204英语二
- 复试：专业课笔试、思想政治理论，综合面试

## 证书授予

完成规定学分、通过学位论文答辩后，授予：
- 东华大学硕士研究生毕业证书
- 工程管理硕士学位证书
