# 中国科学技术大学工程管理硕士（MEM）

## 基本信息

| 项目 | 信息 |
|------|------|
| **院校名称** | 中国科学技术大学 |
| **项目名称** | 工程管理硕士（MEM） |
| **培养学院** | 管理学院 |
| **项目类型** | 非全日制 |
| **学制** | 3年（最长学习年限5年） |
| **学费** | 19.8万元（全程） |
| **上课方式** | 周末授课 |
| **授课地点** | 中国科大校本部（合肥） |
| **院校属性** | 985、211、双一流A类 |
| **官方网址** | https://yz.ustc.edu.cn |

## 项目特色

中国科学技术大学MEM项目充分发挥学校在前沿科学和高新技术领域的优势：

- **理工背景深厚**：依托学校在量子信息、人工智能、深空探测等前沿领域的积累
- **双导师制培养**：校内学术导师+企业实践导师
- **方向设置多元**：项目管理、物流工程与管理、科研和行业工程管理等方向
- **AACSB认证**：管理学院通过国际精英商学院AACSB认证
- **产学研深度融合**：与京东、海康威视、中国邮政等行业代表性企业深度合作

## 招生项目

| 项目方向 | 简介 |
|----------|------|
| 项目管理 | 公共项目决策与管理、科技项目管理、医疗卫生与教育项目管理、工程项目管理 |
| 物流工程与管理 | 物流系统规划与设计、物流企业管理、智慧物流、物流金融 |
| 科研和行业工程管理 | 科研工程管理、能源环境工程、工程质量管理、管理信息化 |

## 报考条件

- 大学本科毕业后有3年以上工作经验
- 高职高专毕业或本科结业后有5年以上工作经验（达到本科毕业同等学力）
- 获硕士、博士学位后有2年以上工作经验

**注**：非全日制仅招收在职定向就业人员

## 考试科目

- 初试：199管理类综合能力 + 204英语二
- 复试：思想政治理论笔试、专业课考核（工程经济学、项目管理等）、综合面试

## 证书授予

完成规定学分、通过学位论文答辩后，授予：
- 中国科学技术大学硕士研究生毕业证书（非全日制）
- 工程管理硕士学位证书
