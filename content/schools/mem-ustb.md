# 北京科技大学工程管理硕士（MEM）

## 基本信息

| 项目 | 信息 |
|------|------|
| **院校名称** | 北京科技大学 |
| **项目名称** | 工程管理硕士（MEM） |
| **培养学院** | 经济管理学院 |
| **项目类型** | 非全日制 |
| **学制** | 2年（最长学习年限5年） |
| **学费** | 9.8万元（全程，4.9万元/年） |
| **上课方式** | 周末授课 |
| **授课地点** | 北京科技大学（北京） |
| **招生人数** | 约15人 |
| **院校属性** | 211，双一流 |
| **官方网址** | https://yzxc.ustb.edu.cn |

## 项目特色

北京科技大学MEM项目依托学校"钢铁"学科优势：

- **工程领域深厚积累**：依托学校在材料、冶金、机械等领域的传统优势
- **大数据与商务分析**：课程涵盖数据分析技术和管理学方法
- **双导师培养**：校内导师+校外导师联合指导
- **校友资源丰富**：定期组织校友行业沙龙、校友大讲堂等活动
- **国际认证衔接**：与相关国际认证机构合作

## 研究方向

| 方向 | 简介 |
|------|------|
| 工程、投资与信息项目管理 | 项目全生命周期管理 |
| 数字管理与智慧运营 | 大数据分析与商务智能 |

## 报考条件

- 大学本科毕业后有3年以上工作经验
- 高职高专毕业或本科结业后有5年以上工作经验
- 获硕士、博士学位后有2年以上工作经验

**注**：仅招收在职定向就业人员

## 考试科目

- 初试：199管理类综合能力 + 204英语二
- 复试：综合素质面试、英语口语、政治笔试

## 证书授予

完成规定学分、通过学位论文答辩后，授予：
- 北京科技大学硕士研究生毕业证书
- 工程管理硕士学位证书
