# 中国传媒大学-英国考文垂学院设计学硕士

## 项目基本信息

| 项目名称 | 内容 |
|---------|------|
| **中方院校** | 中国传媒大学 |
| **外方院校** | 英国考文垂大学 (Coventry University) |
| **专业方向** | 设计学 (Design) |
| **学制** | 2年 |
| **总学费** | 16万元 |
| **是否留服认证** | 是 |
| **上课方式** | 全日制授课 |
| **招生人数** | 50人/年 |
| **入学方式** | 自主招生 |
| **上课地点** | 中国传媒大学（海南） |
| **批准书编号** | 待确认 |

## 项目特色

- **教育部备案项目**：经教育部正式批准的中外合作办学机构
- **海南国际教育创新区**：位于陵水黎安国际教育创新试验区
- **国际化教学**：引入英方先进教学理念和课程体系
- **传媒+设计融合**：结合中国传媒大学传媒学科优势
- **设计学科优势**：考文垂大学艺术设计类学科英国排名前列

## 合作院校简介

### 中国传媒大学
- 教育部直属高校
- 国家"211工程"和"985优势学科创新平台"建设高校
- 国家"双一流"建设高校
- 传媒学科领域顶尖学府

### 考文垂大学
- 英国排名前列的现代化大学
- 设计与视觉文化专业英国领先
- 艺术与设计学院（FADA）国际声誉卓著
- 注重创新和实践的教学理念

## 招生条件

1. 具有中国教育部认可的本科学历
2. 设计、传媒、艺术等相关专业背景优先
3. 提交作品集
4. 通过项目入学考核
5. 英语语言能力证明

## 核心课程

| 模块 | 课程内容 |
|-----|---------|
| 设计理论 | 设计思维、设计研究方法 |
| 数字设计 | 交互设计、用户体验设计 |
| 视觉传达 | 品牌设计、视觉系统 |
| 创新实践 | 设计项目、毕业设计 |

## 毕业获证

- 英国考文垂大学设计学硕士学位证书
- 可申请教育部留学服务中心学历学位认证
- 中国传媒大学结业证书

## 联系方式

- 地址：海南省陵水黎安国际教育创新试验区
