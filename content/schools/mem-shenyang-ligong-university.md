# 沈阳理工大学MEM项目介绍

## 学校简介

沈阳理工大学位于辽宁省沈阳市，是一所以工科为主的综合性大学。学校始建于1948年，在装备制造、汽车工程等领域具有鲜明特色。

## MEM项目简介

沈阳理工大学MEM项目面向装备制造、汽车、机器人等行业培养工程管理人才。

### 招生信息

| 项目 | 内容 |
|------|------|
| 招生院系 | 经济管理学院 |
| 培养方向 | 装备制造项目管理、质量管理工程、供应链管理 |
| 授课方式 | 周末授课 |
| 招生人数 | 约25人 |

### 学制学费

| 项目 | 内容 |
|------|------|
| 学制 | 3年 |
| 学费 | 约3.3万元（全程） |
| 授课安排 | 周末+部分集中 |

## 师资力量

学校工程管理教师团队结构合理，在装备制造领域具有较强的科研和实践能力。

## 就业前景

毕业生主要就业于装备制造企业、汽车工业、军工企业等，就业前景良好。
