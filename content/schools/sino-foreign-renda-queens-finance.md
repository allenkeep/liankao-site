# 中国人民大学-加拿大女王大学金融学硕士

## 项目基本信息

| 项目名称 | 内容 |
|---------|------|
| **中方院校** | 中国人民大学 |
| **外方院校** | 加拿大女王大学 (Queen's University) |
| **专业方向** | 金融学 (Master of Finance) |
| **学制** | 1年 |
| **总学费** | 45.8万元 |
| **是否留服认证** | 是 |
| **上课方式** | 周末集中授课 |
| **招生人数** | 90人/年（每年2期） |
| **入学方式** | 自主招生 |
| **上课地点** | 中国人民大学校内（北京） |
| **批准书编号** | MOE11CA1A20131441N |

## 项目特色

- **顶尖学科背景**：中国人民大学金融学学科排名全国第一
- **学制短效率高**：1年制密集课程，快速获得硕士学位
- **加拿大常春藤盟校**：女王大学与麦吉尔、多伦多大学齐名
- **CFA合作院校**：商学院为CFA考试机构会员单位
- **国际认证**：AACSB和EQUIS双重认证
- **高端人脉**：学员多为金融行业中高层管理者

## 招生条件

1. 经济、金融、统计、会计专业本科毕业，具有学士学位
2. 有意从事金融行业工作的应届毕业生及0-5年工作经验在职人员
3. 有金融或相关学科的学习或工作经历
4. 语言要求：托福网考100分以上或雅思7.0以上
5. 两年以上相关工作经验

## 核心课程

| 课程名称 | 描述 |
|---------|------|
| 公司金融和财务报表分析 I/II | 财务分析与决策 |
| 股票市场与分析 | 权益投资分析 |
| 高级财务模型 | 量化建模实践 |
| 定量方法和经济学 | 数量分析基础 |
| 固定收益产品与市场 | 债券投资与利率风险 |
| 高级投资组合管理 | 组合构建与优化 |
| 另类投资 | PE、对冲基金等 |
| 金融衍生品 | 期权期货定价 |
| 投资银行 | IPO、并购咨询 |
| 金融科技与创新 | 金融科技前沿 |
| 国际金融 | 跨境投资 |

## 毕业获证

- 加拿大女王大学金融学硕士学位证书
- 可申请教育部留学服务中心学历学位认证
- 中国人民大学结业证书
