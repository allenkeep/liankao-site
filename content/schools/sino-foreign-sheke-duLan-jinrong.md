# 中国社会科学院大学-美国杜兰大学金融管理硕士

## 项目基本信息

| 项目名称 | 内容 |
|---------|------|
| **中方院校** | 中国社会科学院大学 |
| **外方院校** | 美国杜兰大学 (Tulane University) |
| **专业方向** | 金融管理 (Master of Finance) |
| **学制** | 1.5年（18个月） |
| **总学费** | 32.8万元 |
| **是否留服认证** | 是 |
| **上课方式** | 周末集中授课 |
| **招生人数** | 60人/年 |
| **入学方式** | 自主招生（材料审核+笔试+面试） |
| **上课地点** | 中国社会科学院大学望京校区 |
| **批准书编号** | MOE11US1A20121203N |

## 项目特色

- **零门槛报考**：不限专业、不限行业背景，本科毕业并取得学位证书即可申请
- **免联考入学**：采用自主招生模式，无需参加全国硕士统考
- **全程不出国**：1.5年学制，周末集中授课，无强制出国要求
- **双名校师资**：社科院顶级金融专家+杜兰大学弗里曼商学院终身教授联合授课
- **高端圈层**：学员90%以上为银行、证券、基金、上市公司中高层管理者
- **权威认证**：毕业获美国杜兰大学金融硕士学位，可申请教育部留学服务中心认证

## 招生条件

1. 具有中华人民共和国国籍
2. 拥护中国共产党领导，品德良好，遵纪守法
3. 具有中国教育部认可的本科学历和学位
4. 具备较好的英语能力（英语四级425分/托福95分/雅思6.5分等）
5. 一年及以上工作经历（特别优秀者可适当放宽）
