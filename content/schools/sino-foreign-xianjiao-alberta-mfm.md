# 西安交通大学-加拿大阿尔伯塔大学金融财务管理硕士

## 项目基本信息

| 项目名称 | 内容 |
|---------|------|
| **中方院校** | 西安交通大学 |
| **外方院校** | 加拿大阿尔伯塔大学 (University of Alberta) |
| **专业方向** | 金融财务管理 (Master of Financial Management) |
| **学制** | 2年（在职学习） |
| **总学费** | 31.8万元 |
| **是否留服认证** | 是 |
| **上课方式** | 周末/集中授课（深圳/上海） |
| **招生人数** | 60人/年 |
| **入学方式** | 自主招生（面试为主） |
| **上课地点** | 深圳异地办学/上海 |
| **批准书编号** | MOE61CA1A20121237N |

## 项目特色

- **QS世界排名前100**：阿尔伯塔大学2026年QS世界大学排名第94位
- **AACSB双重认证**：西安交大管理学院与阿尔伯塔商学院均获AACSB认证
- **QS排名中国第2**：2018年QS全球金融硕士排名第85位，中国第2、亚洲第5
- **免联考入学**：自主招生，英文面试录取
- **中英双语授课**：阿尔伯塔教授英文授课+西安交大教师辅导
- **无需论文**：修满学分即可毕业

## 招生条件

1. 大学本科学历及以上，一般具有学士学位，GPA≥2.7，至少5年以上工作经验
2. 大学本科学历及以上，学士学位，GPA<2.7，至少7年工作经验
3. 无学位申请者，至少15年及以上工作经验，综合评定择优录取
