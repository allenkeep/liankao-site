## 北京语言大学与英国诺丁汉特伦特大学合作举办外国语言学及应用语言学硕士研究生教育项目

| 项目 | 内容 |
|------|------|
| 项目名称 | 北京语言大学与英国诺丁汉特伦特大学合作举办外国语言学及应用语言学硕士研究生教育项目 |
| 办学地址 | 北京市海淀区学院路15号 |
| 中方办学者 | 北京语言大学 |
| 外方办学者 | Nottingham Trent University, UK（英国诺丁汉特伦特大学） |
| 办学层次和类别 | 硕士研究生教育 |
| 学制 | 3年 |
| 每期招生人数 | 45人（在北京语言大学招生规模内统筹安排） |
| 招生起止年份 | 2016年—2030年（每年1期） |
| 招生方式 | 纳入国家硕士研究生招生计划，参加全国硕士研究生统一入学考试 |
| 开设专业或课程 | 外国语言学及应用语言学（专业代码：050211H） |
| 颁发证书（中方） | 北京语言大学外国语言学及应用语言学文学硕士 |
| 颁发证书（外方） | Master of Arts 文学硕士 |
| 审批机关 | 教育部 |
| 批准书编号 | MOE11GB1A20151744N |
| 批准书有效期 | 2033年12月31日 |
