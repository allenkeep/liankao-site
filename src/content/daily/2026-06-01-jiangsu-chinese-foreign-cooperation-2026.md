教育部中外合作办学监管工作信息平台公布最新一批审批结果，江苏省新增7家中外合作办学机构和8个中外合作办学项目，多所将于2026年正式启动招生。

## 新增中外合作办学机构（7个）

| 机构名称 | 合作方 | 办学特点 |
|----------|--------|----------|
| 南京财经大学都柏林学院 | 爱尔兰都柏林大学 | 数字经济、物流管理、审计学 |
| 南京审计大学金审学院国际智慧康养学院 | 韩国世翰大学 | 江苏省独立学院首家中外合作办学机构 |
| 金陵科技学院德蒙福特学院 | 英国德蒙福特大学 | - |
| 南京艺术学院伯恩茅斯国际艺术学院 | 英国伯恩茅斯大学 | - |
| 盐城师范学院爱尔丁学院 | - | - |
| 江苏大学迪肯大学联合学院 | 澳大利亚迪肯大学 | - |
| 江苏第二师范学院迪吉彭学院 | - | - |

## 新增硕士学位项目

**南京邮电大学与俄罗斯国家研究型信息技术、机械与光学大学合作举办电子信息（机器人）专业硕士研究生教育项目**

- 预计2026年9月招生
- 关注两个时间节点：6月底至7月初查看2027年招生专业目录预告，9月份查看正式版本

## 部分项目详情

### 南京财经大学都柏林学院
- **办学规模**：1280人，每年招收320人
- **本科专业**：数字经济（100人）、物流管理（100人）、审计学（120人）

### 南京审计大学金审学院国际智慧康养学院
- **办学规模**：1200人
- **本科专业**：审计学、数据科学与大数据技术、养老服务管理
- **培养模式**：4年"4+0"，全程国内学习，无需出国
