## 北京城市学院与英国华威大学合作举办项目管理硕士学位教育项目

| 项目 | 内容 |
|------|------|
| 项目名称 | 北京城市学院与英国华威大学合作举办项目管理硕士学位教育项目 |
| 办学地址 | 北京城市学院航天城校区，北京市海淀区永丰高科技园区 |
| 中方办学者 | 北京城市学院 |
| 外方办学者 | The University of Warwick, UK（英国华威大学） |
| 办学层次和类别 | 外国硕士学位教育 |
| 学制 | 全日制1.5年，非全日制1.5-4年 |
| 每期招生人数 | 75人 |
| 招生起止年份 | 2014年—2031年（每年2期） |
| 招生方式 | 自主招生（招生标准不低于英国华威大学在英国的标准） |
| 开设专业或课程 | 项目管理 |
| 颁发证书（中方） | 无 |
| 颁发证书（外方） | Master of Science in Programme and Project Management（项目管理专业理学硕士学位证书） |
| 审批机关 | 教育部 |
| 批准书编号 | MOE11UK1A20131445N |
| 批准书有效期 | 2035年12月31日 |
