## 华东理工大学2027级MBA/EMBA招生启动

华东理工大学商学院于2026年5月25日正式发布2027级MBA/EMBA奖助学金方案，同时启动职业成长营报名。

### 项目认证

华理商学院已获得AACSB、EQUIS、AMBA、CAMEA四项国际/国内认证，培养质量获广泛认可。

### 奖助学金方案（12类）

2027级奖助学金方案为奖励优秀学生设立，涵盖以下类型：

| 奖学金项 | 金额/人 | 评选标准 |
|---------|--------|---------|
| 杰出人才奖学金 | 全额学费 | 获省部委级及以上荣誉称号 |
| 统考奖学金 | 一等奖：全额学费 / 二等奖：50%学费 | 一等奖1名，二等奖2名 |
| 职业成长奖学金 | 5万 | 参与职业成长营的学生 |
| 诚意奖学金 | 2万 | 一志愿报考并录取的非全日制学生 |
| 全球人才奖学金 | 2-5万 | QS1-100名5万，101-200名3万，201-300名2万 |
| 校友/亲友奖学金 | 2/3万 | 华理校友或教职员工及其配偶、父母、子女就读 |
| 同心奖学金 | 2/3万 | 由校友/在读生/教职员工推荐并录取 |
| 产教融合精英奖学金 | 2/3万 | 产教融合基地企业员工 |
| 戈奖奖学金 | 2/3万 | 由玄奘之路秘书处开具戈友证明 |
| 飞行奖学金 | 2/3万 | 非上海市学生，需产生住宿和交通费 |
| 科创产业奖学金 | 2万 | 科创MBA班、大健康EMBA班学生 |

### 职业成长营

2027年入学华理MBA/EMBA职业成长营已启动，参与可获5万元职业成长奖学金。营员可提前体验课程、与导师面对面交流，建议有意向的考生尽早报名。

![2027级职业成长营](/img/posts/ecust-mba-2027-5.png)

### 报考信息

- **项目类型**：MBA（全日制/非全日制）、EMBA（非全日制）
- **授课地点**：华东理工大学徐汇校区
- **招生咨询**：详见华东理工大学商学院官网
