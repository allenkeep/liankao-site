## 对外经济贸易大学与法国巴黎第一大学合作举办企业管理硕士学位教育项目

| 项目 | 内容 |
|------|------|
| 项目名称 | 对外经济贸易大学与法国巴黎第一大学合作举办企业管理硕士学位教育项目 |
| 办学地址 | 北京市朝阳区惠新东街10号 |
| 中方办学者 | 对外经济贸易大学 |
| 外方办学者 | L'Universite de Paris 1 Pantheon-Sorbonne, France（法国巴黎第一大学） |
| 办学层次和类别 | 外国硕士学位教育 |
| 学制 | 2年 |
| 每期招生人数 | 90人 |
| 招生起止年份 | 2007年—2026年（每年1期） |
| 招生方式 | 自主招生（招生标准不低于巴黎一大在法国的招生标准：已获得本科毕业证书及学士学位证书，且不少于三年工作经验） |
| 开设专业或课程 | 企业管理 |
| 颁发证书（中方） | 无 |
| 颁发证书（外方） | DIPLÔME D'ÉTUDES SUPÉRIEURES SPÉCIALISÉES（巴黎一大企业管理硕士学位证书） |
| 审批机关 | 教育部 |
| 批准书编号 | MOE11FR1A19970424O |
| 批准书有效期 | 2028年6月27日 |
