# 上海财经大学-美国韦伯斯特大学工商管理硕士(iMBA)

## 项目基本信息

| 项目名称 | 内容 |
|---------|------|
| **中方院校** | 上海财经大学 |
| **外方院校** | 美国韦伯斯特大学 (Webster University) |
| **专业方向** | 工商管理 (MBA) |
| **学制** | 2年（在职学习） |
| **总学费** | 29.8万元 |
| **是否留服认证** | 是 |
| **上课方式** | 周末隔周上课，每月两次 |
| **招生人数** | 150人/年 |
| **入学方式** | 自主招生（笔试+面试） |
| **上课地点** | 上海财经大学商学院（虹口校区） |
| **批准书编号** | MOE31US1A20030513O |

## 项目特色

- **国内首批中外合作MBA项目**：1996年经教育部批准，办学29年
- **AACSB/EQUIS/AMBA三重认证**：上海财经大学商学院获国际顶级商学院认证
- **无需统考**：自主招生，材料审核+笔试+面试录取
- **学分制无论文**：修满学分即可毕业，无毕业论文要求
- **国际交流机会**：可申请赴韦伯斯特大学欧美校区交换学习
- **丰厚奖学金**：综合排名第1名可获10万元奖学金

## 招生条件

1. 拥护中国共产党的领导，遵纪守法，政审合格
2. 本科或三年全日制专科学历，GPA≥2.0
3. 三年及以上全职工作经验

## 免试通道

符合以下条件可申请免笔试直接面试：
- 参加往年管理类硕士联考，管综和英语均超过国家线A线
- 全英文授课的全日制本科毕业生
- GMAT成绩600分以上
- 已获得研究生学历或硕士学位

## 核心课程

- **思政模块**：中国特色社会主义理论与实践研究
- **基础模块**：商业当代经济分析、会计理论与实务、管理信息系统
- **核心课程**：公司金融、组织行为学、管理经济学、运营管理
- **财务方向**：投资学、国际金融、行为金融、财务报表分析、并购与重组
- **营销方向**：消费者行为学、电子商务、国际营销、数字营销

## 毕业获证

- 美国韦伯斯特大学工商管理硕士（MBA）学位证书
- 中国教育部留学服务中心国外学历学位认证书
- 美国韦伯斯特大学方向证书
- 上海财经大学结业证书
