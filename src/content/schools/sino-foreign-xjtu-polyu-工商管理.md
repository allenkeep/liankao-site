## 项目简介
西安交通大学与香港理工大学合作举办工商管理，经教育部批准设立（批准书编号：MOE61HK1A20020394O）。项目免全国联考，采用申请制入学，毕业获外方硕士学位，可在中国教育部留学服务中心进行学历学位认证。

## 审批信息

| 项目信息 | 详情 |
|---------|------|
| 中方办学者 | 西安交通大学 |
| 外方办学者 | 香港理工大学 |
| 开设专业 | 工商管理 |
| 批准书编号 | MOE61HK1A20020394O |
| 学制 | 2年 |
| 每期招生人数 | 150人 |
| 招生起止年份 | 2002年—2028年 |
| 招生方式 | 自主招生 |
| 学费 | 30-34.9万元 |
| 学历认证 | 可留服认证 |

## 申请条件
- 本科及以上学历
- 相关专业背景优先
- 具备一定英语能力
- 具体条件以当年招生简章为准

## 项目特色
- 教育部批准，学历可留服认证
- 免全国联考，申请制入学
- 中外双方师资联合授课
- 国际化课程体系
