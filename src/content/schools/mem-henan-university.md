# 河南大学MEM项目介绍

## 学校简介

河南大学坐落在历史文化名城、八朝古都开封，是国家"双一流"建设高校、河南省人民政府与教育部共建高校。学校商科起源于1927年的国立第五中山大学。

## MEM项目简介

河南大学MEM项目由商学院与建筑工程学院联合承办，依托管理科学与工程学科优势，培养懂技术、懂管理、懂经济的复合型高级工程管理人才。

### 研究领域

- **制造工程**：面向装备制造行业，解决产品设计、流程再造、计划与控制等问题
- **项目管理**：面向大型基础设施、新能源开发等复杂工程，关注项目投融资、成本控制、风险管理等
- **大数据与商务智能**：面向医疗、物流、金融等服务业，研究数据分析与智能决策

### 招生信息

| 项目 | 内容 |
|------|------|
| 招生院系 | 商学院、建筑工程学院 |
| 培养方向 | 工程项目管理、工程经济与造价管理、大数据与信息工程管理 |
| 授课方式 | 周末班 |
| 招生人数 | 约15人 |

### 学制学费

| 项目 | 内容 |
|------|------|
| 学制 | 3年（最长5年） |
| 学费 | 6万元（全程，2万/年） |
| 授课地点 | 开封金明校区 |

## 师资力量

现有专任指导教师22人，其中教授5人、副教授11人、博士生导师5人；在设备制造、医药制造、土木工程、物流与供应链等多个行业拥有企业职业导师10人。

## 报考条件

- 专科毕业后5年及以上工作经验
- 本科毕业后3年及以上工作经验
- 硕士/博士学位后2年及以上工作经验

## 就业前景

毕业生主要就业于大型企业、设计院、咨询公司、政府部门等。
