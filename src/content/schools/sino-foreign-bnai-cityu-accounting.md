# 北京国家会计学院-香港城市大学智能会计与金融科技硕士

## 项目基本信息

| 项目名称 | 内容 |
|---------|------|
| **中方院校** | 北京国家会计学院 |
| **外方院校** | 香港城市大学 (City University of Hong Kong) |
| **专业方向** | 智能会计与金融科技应用 (MSc Intelligent Accounting and FinTech) |
| **学制** | 2年 |
| **总学费** | 29.8万元 |
| **是否留服认证** | 是 |
| **上课方式** | 中文授课，周末为主 |
| **招生人数** | 60人/年 |
| **入学方式** | 自主招生 |
| **上课地点** | 北京国家会计学院（顺义） |
| **批准书编号** | MOE11HK1A20212101N |

## 项目特色

- **QS世界排名前65位**：香港城市大学2025年QS世界大学排名第62位
- **中文授课无压力**：采用中文授课为主，无语言障碍
- **会计+科技融合**：培养复合型智能会计人才
- **不用统考**：自主招生，材料审核+面试录取
- **财政部直属院校合作**：北京国家会计学院是财政部直属事业单位
- **不出国门获认证**：全程在国内完成学习

## 香港城市大学简介

- 2025年QS世界大学排名第62位
- 2025年QS亚洲大学排名第23位
- 赛马会动物医学及生命科学院等学科世界领先
- AACSB和EQUIS双重认证

## 北京国家会计学院简介

- 国家财政部直属事业单位
- 中国顶尖CFO摇篮
- 国家会计专业质量认证（AAPEQ）A级成员单位
- 教学管理依托清华大学

## 招生条件

1. 本科及以上学历（金融财务相关背景优先）
2. 5年及以上工作经验
3. 对会计、金融科技领域有兴趣
4. 通过项目入学考核

## 核心课程

| 模块 | 课程内容 |
|-----|---------|
| 会计基础 | 财务报表分析、管理会计 |
| 金融科技 | 金融科技应用、区块链与金融 |
| 智能分析 | 大数据分析、人工智能应用 |
| 审计控制 | 智能审计、内部控制 |
| 综合实践 | 案例分析、毕业项目 |

## 毕业获证

- 香港城市大学智能会计与金融科技应用硕士学位证书
- 可申请教育部留学服务中心学历学位认证
- 北京国家会计学院结业证书
