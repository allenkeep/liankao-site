## 项目简介
上海交通大学与新加坡国立大学合作举办数量金融，经教育部批准设立（批准书编号：MOE31SG1A20171857N）。项目免全国联考，采用申请制入学，毕业获外方硕士学位，可在中国教育部留学服务中心进行学历学位认证。

## 审批信息

| 项目信息 | 详情 |
|---------|------|
| 中方办学者 | 上海交通大学 |
| 外方办学者 | 新加坡国立大学 |
| 开设专业 | 数量金融 |
| 批准书编号 | MOE31SG1A20171857N |
| 学制 | 2年 |
| 每期招生人数 | 60人 |
| 招生起止年份 | 2018年—2026年 |
| 招生方式 | 自主招生 |
| 学费 | 约77万元（141,800新币） |
| 学历认证 | 可留服认证 |

## 申请条件
- 本科及以上学历
- 相关专业背景优先
- 具备一定英语能力
- 具体条件以当年招生简章为准

## 项目特色
- 教育部批准，学历可留服认证
- 免全国联考，申请制入学
- 中外双方师资联合授课
- 国际化课程体系
