# 北京交通大学工程管理硕士（MEM）

## 基本信息

| 项目 | 信息 |
|------|------|
| **院校名称** | 北京交通大学 |
| **项目名称** | 工程管理硕士（MEM） |
| **培养学院** | 经济管理学院 |
| **项目类型** | 非全日制 |
| **学制** | 2年（最长学习年限5年） |
| **学费** | 12.6万元（全程，6.3万元/年） |
| **上课方式** | 集中班或周末班 |
| **授课地点** | 北京交通大学校本部（北京） |
| **招生人数** | 非全日制约110人 |
| **院校属性** | 211，双一流 |
| **官方网址** | https://yzb.bjtu.edu.cn |

## 项目特色

北京交通大学MEM项目依托学校"交通"特色：

- **四大研究方向**：土木工程管理、信息工程管理、安全工程管理、物流工程管理
- **工管交叉融合**：结合学校在铁路、电信等领域的传统优势
- **案例教学**：课堂教学、专题讲座、企业家论坛相结合
- **校外专家授课**：聘请具有实践经验的校外高级技术职称专家授课
- **A类学科支撑**：管理科学与工程、工商管理、应用经济学获评A类学科

## 研究方向

| 方向 | 简介 |
|------|------|
| 土木工程管理 | 土木工程项目策划决策、投融资、施工安装、竣工交付等全过程管理 |
| 信息工程管理 | 信息系统工程的规划、建模、平台建设、系统集成等全过程管理 |
| 安全工程管理 | 安全系统工程、安全设计与生产、安全测量与评价等 |
| 物流工程管理 | 物流系统规划、物流与供应链管理优化、物流信息系统 |

## 报考条件

- 大学本科毕业后有3年以上工作经验
- 高职高专毕业或本科结业后有5年以上工作经验
- 获硕士、博士学位后有2年以上工作经验

**注**：仅招收在职定向就业人员

## 考试科目

- 初试：199管理类综合能力 + 204英语二
- 复试：综合素质面试、英语口语、政治笔试

## 证书授予

完成规定学分、通过学位论文答辩后，授予：
- 北京交通大学硕士研究生毕业证书
- 工程管理硕士学位证书
