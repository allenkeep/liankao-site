# 电子科技大学-美国韦伯斯特大学工商管理硕士(iMBA)

## 项目基本信息

| 项目名称 | 内容 |
|---------|------|
| **中方院校** | 电子科技大学 |
| **外方院校** | 美国韦伯斯特大学 (Webster University) |
| **专业方向** | 工商管理 (MBA) |
| **学制** | 2年（最长可延至4年） |
| **总学费** | 28.8万元 |
| **是否留服认证** | 是 |
| **上课方式** | 周末上课（周六为主） |
| **招生人数** | 135人/年 |
| **入学方式** | 自主招生 |
| **上课地点** | 电子科技大学经济与管理学院（成都） |
| **批准书编号** | MOE51US1A20020160O |

## 项目特色

- **西南首个中外合作MBA**：中国西南地区首个经教育部认证的中外合作MBA项目
- **985高校合作**：电子科技大学是国家"985"和"211"重点大学
- **AMBA国际认证**：电子科技大学经济与管理学院是中国西部第一所获得AMBA认证的学院
- **BGA认证**：已获BGA国际认证
- **学分制灵活**：最长可延至4年完成学业
- **国际校友网络**：可接入韦伯斯特全球校友网络

## 电子科技大学简介

- 教育部直属全国重点大学
- 国家"211工程""985工程"和首批"双一流"建设高校
- 被称为"中国电子类院校排头兵"
- 电子科学与技术、信息与通信工程等学科实力顶尖

## 韦伯斯特大学简介

- 1915年创办于美国密苏里州圣路易斯市
- 美国中西部顶尖区域大学
- U.S. News排名前25%
- 商科获ACBSP认证

## 招生条件

1. 拥护中国共产党的领导，遵纪守法，品行端正，身心健康
2. 本科及以上学历，毕业后具有5年以上工作经验
3. 专科学历，毕业后具有8年以上工作经验
4. 已累计修满108学分及以上
5. 大中型企业、事业单位中高层管理者优先

## 核心课程

- 管理经济学
- 组织行为学
- 营销管理
- 公司金融
- 战略管理
- 运营管理
- 人力资源管理
- 国际商务
- 创业管理
- 大数据与商务分析

## 毕业获证

- 美国韦伯斯特大学工商管理硕士（MBA）学位证书
- 可申请教育部留学服务中心学历学位认证
- 电子科技大学硕士结业证书
