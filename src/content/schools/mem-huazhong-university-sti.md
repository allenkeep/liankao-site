# 华中科技大学工程管理硕士（MEM）

## 基本信息

| 项目 | 信息 |
|------|------|
| **院校名称** | 华中科技大学 |
| **项目名称** | 工程管理硕士（MEM） |
| **培养学院** | 管理学院、土木与水利工程学院 |
| **项目类型** | 非全日制 |
| **学制** | 3年 |
| **学费** | 管理学院16.8万元，土木与水利工程学院13.5万元（全程） |
| **上课方式** | 周末授课+集中授课 |
| **授课地点** | 华中科技大学主校区（武汉） |

| **官方网址** | https://www.hust.edu.cn |

## 学院与方向设置

### 管理学院
- 工程项目管理
- 研发与创新管理
- 信息系统与信息管理
- 制造与服务运营管理

### 土木与水利工程学院
- 智能建造与管理
- 土木工程管理
- 水利工程管理

## 项目特色

- **光电信息强校**：在机械、光电信息等领域实力雄厚
- **工管融合**：依托学校工科优势，结合管理学院优质教学资源
- **产学研平台**：武汉·中国光谷所在地，高新技术企业资源丰富
- **中部崛起机遇**：武汉是国家中心城市，工程项目建设活跃
- **国际认证**：管理学院通过AACSB国际认证

## 课程设置

### 核心课程
- 工程管理导论
- 系统工程
- 工程经济学
- 项目管理
- 质量管理
- 风险管理
- 管理信息系统

### 方向选修课
根据不同学院方向特色开设专业课程

## 报考条件

1. 大学本科毕业后有3年以上工作经验
2. 获得国家承认的高职高专毕业学历或大学本科结业后，有5年以上工作经验
3. 获得硕士、博士学位后有2年以上工作经验

## 初试科目

| 科目 | 满分 |
|------|------|
| 管理类联考综合能力（199） | 200分 |
| 英语二（204） | 100分 |

## 证书授予

完成培养方案规定学分、通过学位论文答辩，经华中科技大学学位评定委员会审核批准，可获得：
- 华中科技大学硕士研究生毕业证书（非全日制）
- 工程管理硕士（MEM）学位证书
