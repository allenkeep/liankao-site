# 长江大学MEM项目介绍

## 学校简介

长江大学位于湖北省荆州市，是湖北省重点建设的骨干高校。学校学科门类齐全，涵盖理、工、农、医、文、法、经、管、教、艺术等学科。

## MEM项目简介

长江大学城市建设学院工程管理硕士（MEM）项目培养具备工程项目管理能力的专业人才。

### 研究方向

- 工程项目管理
- 土木工程建造与管理

### 招生信息

| 项目 | 内容 |
|------|------|
| 招生院系 | 城市建设学院 |
| 培养方向 | 工程项目管理、土木工程建造与管理 |
| 授课方式 | 周末授课 |
| 招生人数 | 约10人 |

### 学制学费

| 项目 | 内容 |
|------|------|
| 学制 | 3年 |
| 学费 | 约4.5万元（全程） |
| 授课地点 | 湖北荆州 |

## 报考条件

- 本科毕业后有3年以上工作经验
- 高职高专毕业后有5年以上工作经验
- 硕士/博士学位后有2年以上工作经验

## 就业前景

毕业生主要就业于建筑企业、房地产公司、设计院等。
