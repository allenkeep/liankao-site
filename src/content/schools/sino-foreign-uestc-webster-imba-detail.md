## 项目简介

电子科技大学与美国韦伯斯特大学(Webster University)合作的iMBA项目，成都授课，免联考申请制，毕业可留服认证。

## 审批信息

| 项目 | 详情 |
|-----|------|
| 办学地址 | 四川省成都市高新区（西区）西源大道2006号 |
| 中外方办学者 | 中方：电子科技大学；外方：美国韦伯斯特大学 |
| 办学层次 | 外国硕士学位教育 |
| 学制 | 2年 |
| 每期招生人数 | 135人 |
| 招生起止年份 | 2002年—2027年（每年1期） |
| 招生方式 | 自主招生（招生标准不低于韦伯斯特大学在美国的标准） |
| 开设专业 | 工商管理硕士 |
| 颁发证书 | 中方：无；外方：韦伯斯特大学工商管理硕士 |
| 审批机关 | 教育部 |
