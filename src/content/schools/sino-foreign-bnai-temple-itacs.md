# 北京国家会计学院-美国天普大学IT审计与网络安全硕士

## 项目基本信息

| 项目名称 | 内容 |
|---------|------|
| **中方院校** | 北京国家会计学院 |
| **外方院校** | 美国天普大学 (Temple University) |
| **专业方向** | IT审计与网络安全 (MS in IT Auditing and Cyber Security) |
| **学制** | 2年 |
| **总学费** | 26.8万元 |
| **是否留服认证** | 是 |
| **上课方式** | 周末/集中授课 |
| **招生人数** | 40人/年 |
| **入学方式** | 自主招生（材料审核+面试） |
| **上课地点** | 北京国家会计学院（顺义） |
| **批准书编号** | MOE11US1A20151743N |

## 项目特色

- **国内首个IT审计硕士**：填补国内信息技术审计与网络安全相关专业硕士学位空白
- **CISA认证直通**：课程已获ISACA认证，涵盖CISA考试所有模块
- **双院师资**：中美资深教授及实务界精英联袂执教
- **不出国门获认证**：获得海外高校学位，可办理学历认证
- **就业前景广阔**：可入职国际知名会计师事务所、央国企、金融机构

## 招生条件

1. 具有本科及以上学历
2. 会计、审计、信息管理、信息系统安全、IT治理等相关从业者
3. 有志于成为CISA注册信息系统审计师
4. 通过中美双方联合面试

## 核心课程

| 课程名称 | 主要内容 |
|---------|---------|
| IT审计组织治理 | IT治理、风险管理、业务连续性 |
| ERP系统审计 | SAP系统审计方法与实践 |
| 商业信息资产保护 | 信息安全风险管理、CIA要素 |
| IT审计师商业技能 | 沟通、领导力、情商 |
| IT审计师数据分析 | 数据分析、ACL审计工具 |
| IT审计流程 | IT审计生命周期、风险导向审计 |
| 系统与架构生命周期 | 信息系统架构设计与管理 |
| IT审计监管与控制 | COBIT框架、控制评估 |
| 信息安全体系架构 | 安全体系设计、企业安全规划 |
| IT审计能力提升 | CISA考试备考 |

## 毕业获证

- 美国天普大学IT审计与网络安全硕士学位证书
- 可申请教育部留学服务中心学历学位认证
- 可直接报名参加CISA考试
