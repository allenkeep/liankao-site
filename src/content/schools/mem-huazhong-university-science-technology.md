# 华中科技大学MEM项目介绍

## 学校简介

华中科技大学是教育部直属的全国重点大学，国家"双一流"建设高校。学校学科齐全，涵盖哲学、经济学、法学、教育学、文学、理学、工学、医学、管理学、艺术学等学科门类。

## MEM项目简介

华中科技大学MEM项目由土木与水利工程学院和管理学院共同承办，依托学校强大的工科背景和管理学科优势，培养高层次的工程管理人才。

### 培养方向

**土木与水利工程学院方向：**
- 工程项目管理
- 工程经济与造价管理
- 房地产开发与管理
- 基础设施建设与管理

**管理学院方向：**
- 项目管理
- 数智化企业管理
- 精益运营管理
- 智慧供应链管理
- 工程管理

### 招生信息

| 项目 | 内容 |
|------|------|
| 招生院系 | 土木与水利工程学院、管理学院 |
| 学习方式 | 非全日制 |
| 授课方式 | 周末班/集中班 |
| 招生人数 | 参考约50-60人 |

### 学制学费

| 项目 | 内容 |
|------|------|
| 学制 | 3年（最长5年） |
| 学费 | 土木与水利工程学院全程13.5万元，管理学院全程16.8万元 |
| 授课安排 | 分学年缴纳 |

## 培养特色

- **顶尖师资**：汇聚全校优质教育资源，师资力量雄厚
- **学科交叉**：依托土木工程、水利工程、管理科学与工程等优势学科
- **实践导向**：注重理论与实践结合，强调案例教学
- **双导师制**：配备学术导师和行业导师双重指导

## 就业前景

毕业生主要就业于大型国有企业、跨国公司、政府部门、设计院等，就业前景广阔。
