# 天津商业大学-法国Excelia商学院可持续金融硕士

## 项目基本信息

| 项目名称 | 内容 |
|---------|------|
| **中方院校** | 天津商业大学 |
| **外方院校** | 法国Excelia商学院 (Excelia Business School) |
| **专业方向** | 可持续金融 (MSc in Sustainable Finance) |
| **学制** | 2年 |
| **总学费** | 17.6万元 |
| **是否留服认证** | 是 |
| **上课方式** | 周末班/集中班 |
| **招生人数** | 120人/年 |
| **入学方式** | 自主招生 |
| **上课地点** | 天津商业大学 |
| **批准书编号** | MOE12FR1A20232301N |

## 项目特色

- **2024年新获批项目**：教育部批准的中外合作办学项目
- **三大国际认证**：EQUIS、AACSB、AMBA全球精英商学院三重认证
- **服务双碳战略**：响应国家绿色低碳发展战略
- **2+0培养模式**：全程在天津就读，无需出国
- **法语高排名**：Excelia金融类硕士2025年QS排名第121-130位

## Excelia商学院简介

- 法国精英大学校成员
- 连续7年上榜《金融时报》全球最佳商学院（2024年全球第52位）
- 全球不到1%商学院拥有三重认证
- EQUIS和AACSB双重认证

## 招生条件

1. 具有中国教育部认可的本科学历
2. 有一定工作经验者优先
3. 对可持续金融领域有学习兴趣
4. 通过项目入学考核

## 核心课程

- 可持续金融理论
- ESG投资分析
- 绿色金融产品
- 碳市场与碳金融
- 气候变化经济学
- 企业社会责任
- 金融风险管理
- 国际金融

## 毕业获证

- 法国Excelia商学院可持续金融理学硕士学位证书
- 可申请教育部留学服务中心学历学位认证
- 天津商业大学结业证书
