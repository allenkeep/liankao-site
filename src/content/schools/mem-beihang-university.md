# 北京航空航天大学工程管理硕士（MEM）

## 基本信息

| 项目 | 信息 |
|------|------|
| **院校名称** | 北京航空航天大学 |
| **项目名称** | 工程管理硕士（MEM） |
| **培养学院** | 经济管理学院 |
| **项目类型** | 非全日制 |
| **学制** | 2年 |
| **学费** | 16.6万元（全程） |
| **上课方式** | 周末授课 |
| **授课地点** | 北京航空航天大学学院路校区 |
| **招生人数** | 参考往年计划 |
| **复试线** | 参考北京航空航天大学自主划线 |
| **院校属性** | 985、211、双一流A类 |
| **官方网址** | https://www.buaa.edu.cn |

## 项目方向

- 项目管理
- 工程经济与管理
- 质量与可靠性管理
- 航空航天工程管理
- 互联网与大数据工程管理

## 项目特色

- **国防七子之一**：工科和管理学专业都很受欢迎
- **空天信融合**：依托学校在航空航天、信息科学等领域的优势
- **企业研学特色**：组织学生走进小米汽车"黑灯工厂"、北京地铁28号线等标杆企业
- **访学交流**：组织杭州访学等异地研学活动
- **学科竞赛**：组织学生参加中国研究生工程管理案例大赛并获奖
- **实践导向**：注重案例教学和企业参访

## 课程设置

### 核心课程
- 工程管理导论
- 系统工程
- 工程经济学
- 项目管理
- 质量管理
- 可靠性工程
- 风险管理

### 方向选修课
- 航空航天概论
- 大数据分析
- 互联网项目管理
- 质量管理与六西格玛

## 报考条件

1. 大学本科毕业后有3年以上工作经验
2. 获得国家承认的高职高专毕业学历或大学本科结业后，有5年以上工作经验
3. 获得硕士、博士学位后有2年以上工作经验

## 初试科目

| 科目 | 满分 |
|------|------|
| 管理类联考综合能力（199） | 200分 |
| 英语二（204） | 100分 |

## 证书授予

完成培养方案规定学分、通过学位论文答辩，经北京航空航天大学学位评定委员会审核批准，可获得：
- 北京航空航天大学硕士研究生毕业证书（非全日制）
- 工程管理硕士（MEM）学位证书
