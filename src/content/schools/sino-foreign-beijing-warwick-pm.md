# 北京城市学院-英国华威大学项目管理硕士

## 项目基本信息

| 项目名称 | 内容 |
|---------|------|
| **中方院校** | 北京城市学院 |
| **外方院校** | 英国华威大学 (The University of Warwick) |
| **专业方向** | 项目管理 (MSc in Programme and Project Management) |
| **学制** | 非全日制1.5-4年/全日制1.5年 |
| **总学费** | 20万元 |
| **是否留服认证** | 是 |
| **上课方式** | 周末集中授课（连续两周周五晚+周六日） |
| **招生人数** | 每班≤35人 |
| **入学方式** | 申请审核制（材料审核+面试） |
| **上课地点** | 北京城市学院航天城校区 |
| **批准书编号** | MOE11UK1A20131445N |

## 项目特色

- **QS世界排名前75位**：华威大学2026年QS世界大学排名第74位
- **纯正英式教学**：所有核心课程由华威大学WMG教授全程担纲
- **IPMP职业资质直通**：课程覆盖IPMA知识体系，毕业可直接申请IPMP D级
- **两次赴英机会**：可赴华威校本部及全球海外教学中心体验学习
- **企业高管案例**：每门课程特设知名企业高管实战案例研讨

## 招生条件

### 优秀学术类（毕业4年内）
- 985/211/双一流本科毕业并获学士学位
- 雅思≥6.5分/托福≥92分/PTE≥62分/六级≥495分

### 实践经验类（毕业4年+工作经验）
- 普通本科毕业并获学士学位，专业不限
- 由任职企业出具英语水平证明

## 课程体系

### 英方授课模块
- Project Planning, Management & Control（项目计划管理与控制）
- Project Financial Management（项目财务管理）
- Programme and Project Strategy（项目战略）
- International Business Development（国际商业发展）
- Interdisciplinary Research Methods（跨学科研究方法）
- Global Marketing Strategy（全球营销战略）
- Management of Change（变革管理）
- Managing the Multi-Project Environment（复合项目管理）

### 中方授课模块
- 项目可行性研究与后评估
- 产品设计与开发
- IPMP能力与素养培养
- 新时代中国特色社会主义理论与实践研究
- 自然辩证法
- 毕业论文

## 毕业获证

- 英国华威大学项目管理理学硕士学位
- 可申请IPMP D级证书（C/B/A级免笔试）
- 可办理中国教育部留学服务中心学历认证

## 联系方式

- 申请费：1800元
- 预科费：1万元（非必需）
