# 重庆大学工程管理硕士（MEM）

## 基本信息

| 项目 | 信息 |
|------|------|
| **院校名称** | 重庆大学 |
| **项目名称** | 工程管理硕士（MEM） |
| **培养学院** | 管理科学与房地产学院 |
| **项目类型** | 非全日制 |
| **学制** | 3年 |
| **学费** | 12.9万元（43000元/生·年） |
| **上课方式** | 周末授课为主 |
| **授课地点** | 重庆大学A/B区 |
| **招生人数** | 300人（不含推免） |
| **院校属性** | 985、211、双一流A类 |
| **官方网址** | https://graduate.cqu.edu.cn |

## 项目特色

重庆大学MEM项目充分发挥学校在建筑、土木工程等领域的学科优势：

- **学科优势突出**：管理科学与房地产学院在教育部学科评估中表现优异
- **建筑土木特色**：依托学校建筑老八校背景，聚焦房地产开发与项目管理
- **师资力量雄厚**：汇聚校内外知名学者和行业专家
- **实践导向培养**：课程设置紧密结合工程建设实际需求
- **资源平台优越**：与众多大型企业建立合作关系

## 招生方向

| 研究方向 | 简介 |
|----------|------|
| 建设工程管理 | 工程项目决策与全过程管理、房地产开发与经营 |
| 物流工程与管理 | 供应链管理、物流系统优化、仓储与配送管理 |
| 工业工程与管理 | 生产系统优化、质量管理、供应链管理 |
| 信息工程管理 | 信息系统项目管理、电子商务、大数据分析 |

## 报考条件

- 大学本科毕业后有3年以上工作经验
- 高职高专毕业或本科结业后有5年以上工作经验（达到本科毕业同等学力）
- 获硕士、博士学位后有2年以上工作经验

**注**：非全日制仅招收在职定向就业人员，录取前须签订定向培养协议

## 考试科目

- 初试：199管理类综合能力 + 204英语二
- 复试：思想政治理论、外语听力口语、专业综合知识（经济学、管理学、工程经济与管理等）
- 同等学力加试：管理学、经济学

## 证书授予

完成规定学分、通过学位论文答辩后，授予：
- 重庆大学硕士研究生毕业证书（非全日制）
- 工程管理硕士学位证书
