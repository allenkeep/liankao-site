# 上海大学工程管理硕士（MEM）

## 基本信息

| 项目 | 信息 |
|------|------|
| **院校名称** | 上海大学 |
| **项目名称** | 工程管理硕士（MEM） |
| **培养学院** | 管理学院 |
| **项目类型** | 非全日制 |
| **学制** | 2.5年 |
| **学费** | 15.8万元（全程） |
| **上课方式** | 周末上课 |
| **授课地点** | 上海大学宝山校区 |
| **招生人数** | 约120人 |
| **院校属性** | 211，双一流 |
| **官方网址** | https://ms.shu.edu.cn |

## 项目特色

上海大学MEM项目依托管理学院的学科优势：

- **IPMP认证合作**：2020年取得IPMP认证合作资质
- **企业真实项目**：基于企业真实项目和环境的案例教学
- **系统知识整合**：对知识的系统把握与实践经验整合
- **高度互动教学**：采用互动式教学方法
- **两院院士资源**：历史上曾有刘人怀院士、刘源张院士担任院长

## 研究方向

| 方向 | 简介 |
|------|------|
| 建设工程管理 | 工程项目全生命周期管理 |
| 制造工程管理 | 制造业项目管理与优化 |
| 产品开发工程管理 | 新产品开发项目管理 |

## 报考条件

- 大学本科毕业后有3年以上工作经验
- 高职高专毕业或本科结业后有5年以上工作经验
- 获硕士、博士学位后有2年以上工作经验

**注**：仅招收在职定向就业人员

## 考试科目

- 初试：199管理类综合能力 + 204英语二
- 复试：专业课笔试、思想政治理论，综合面试

## 证书授予

完成规定学分、通过学位论文答辩后，授予：
- 上海大学硕士研究生毕业证书
- 工程管理硕士学位证书
