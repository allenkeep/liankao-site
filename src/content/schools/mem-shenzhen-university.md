# 深圳大学MEM项目介绍

## 学校简介

深圳大学位于中国改革开放的前沿城市深圳，是一所在改革开放中崛起的高水平大学。学校在粤港澳大湾区建设中具有重要地位，拥有优越的区位优势和丰富的产学研资源。

## MEM项目简介

深圳大学工程管理硕士（MEM）项目紧密对接粤港澳大湾区建设和深圳"20+8"产业集群发展需求，面向土木建筑、交通运输、智能制造等关键行业培养高层次工程管理人才。

### 研究方向

- 土木工程管理
- 智能建造与智慧交通
- 工程项目投融资与造价管理
- 工程信息管理

### 招生信息

| 项目 | 内容 |
|------|------|
| 招生院系 | 土木与交通工程学院 |
| 学习方式 | 非全日制 |
| 授课方式 | 周末授课 |
| 招生人数 | 约30人 |

### 学制学费

| 项目 | 内容 |
|------|------|
| 学制 | 3年（最长5年） |
| 学费 | 约9万元（全程） |
| 授课地点 | 深圳大学沧海校区/粤海校区 |

## 项目特色

- **区位优势**：地处粤港澳大湾区核心引擎深圳
- **工管融合**：强调工程技术与管理科学的交叉融合
- **双导师制**：校内学术导师+行业实践导师
- **实践导向**：案例教学、项目实训、企业参访

## 师资力量

学院师资力量雄厚，在土木工程、交通工程领域具有较强的科研实力。

## 就业前景

毕业生主要就业于粤港澳大湾区的建筑企业、房地产公司、科技企业等。
