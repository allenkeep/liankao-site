# 长沙理工大学MEM项目介绍

## 学校简介

长沙理工大学是一所以工科为主，理、工、管、文、经、法、艺等多学科协调发展的多科性大学。学校在交通运输工程、土木工程等领域具有鲜明特色和明显优势。

## MEM项目简介

长沙理工大学工程管理硕士（MEM）项目由交通运输工程学院承办，依托学校在交通运输、土木工程等领域的学科优势，培养工程管理领域的专业人才。

### 研究方向

- 工程项目管理
- 工程经济与项目评价
- 公路运营管理

### 招生信息

| 项目 | 内容 |
|------|------|
| 招生院系 | 交通运输工程学院 |
| 学习方式 | 非全日制 |
| 授课方式 | 周末授课 |
| 招生人数 | 约20人 |

| 项目 | 内容 |
|------|------|
| 学制 | 3年 |
| 学费 | 约5.4万元（全程） |
| 授课地点 | 湖南长沙 |

## 报考条件

- 本科毕业后有3年以上工作经验
- 高职高专毕业后有5年以上工作经验
- 硕士/博士学位后有2年以上工作经验

## 师资力量

学院在交通运输工程领域拥有一批实践经验丰富的教师团队，在公路工程、交通规划等领域具有较强的科研能力。

## 就业前景

毕业生主要就业于交通运输企业、建筑企业、设计院、政府交通管理部门等。
