# 吉林大学工程管理硕士（MEM）

## 基本信息

| 项目 | 信息 |
|------|------|
| **院校名称** | 吉林大学 |
| **项目名称** | 工程管理硕士（MEM） |
| **培养学院** | 经济学院 |
| **项目类型** | 非全日制 |
| **学制** | 3年（最长修业年限5年） |
| **学费** | 长春班7.8万元/广东班10.98万元（全程） |
| **上课方式** | 双休日或节假日集中授课 |
| **授课地点** | 长春本部及广东实践基地 |
| **招生人数** | 约150人 |
| **院校属性** | 985、211、双一流A类 |
| **官方网址** | https://jjxy.jlu.edu.cn |

## 项目特色

吉林大学MEM项目依托经济学院的学科优势：

- **四大核心方向**：智能制造、供应链管理、金融科技、企业工程管理
- **产学研协同**：与中咨公司、中航工业等企业共建实践基地
- **多学科交叉**：融合管理、金融、智能制造、人工智能等前沿课程
- **异地培养**：广东班学员五一、十一假期返校集中学习
- **案例教学**：案例教学资源入选"全国百篇优秀管理案例"

## 培养方向

| 方向 | 核心能力 |
|------|----------|
| 智能制造 | 智能制造技术应用、全周期工程管理、创新与领导力 |
| 供应链管理 | 供应链全流程规划、数据驱动决策、跨领域协同管理 |
| 金融科技 | 金融工程实践能力、技术创新与应用、复杂问题解决 |
| 企业工程管理 | 工程全周期管理、国际化视野、多学科融合应用 |

## 报考条件

- 大学本科毕业后有3年以上工作经验
- 高职高专毕业或本科结业后有5年以上工作经验
- 获硕士、博士学位后有2年以上工作经验

**注**：仅招收在职定向就业人员

## 考试科目

- 初试：199管理类综合能力 + 204英语二
- 复试：政治笔试（开卷）、管理案例分析（闭卷）、英语口语、综合面试

## 证书授予

完成规定学分、通过学位论文答辩后，授予：
- 吉林大学硕士研究生毕业证书（注明"非全日制"）
- 工程管理硕士学位证书
