# 上海大学-悉尼科技大学工程管理/金融硕士

## 项目基本信息

| 项目名称 | 内容 |
|---------|------|
| **中方院校** | 上海大学 |
| **外方院校** | 澳大利亚悉尼科技大学 (University of Technology Sydney) |
| **专业方向** | 工程管理硕士(MEM)/金融硕士(MF) |
| **学制** | 1.5年 |
| **总学费** | MEM: 20.8万元 / MF: 21.8万元 |
| **是否留服认证** | 是 |
| **上课方式** | 周末上课（周六、周日） |
| **招生人数** | 各75人/年 |
| **入学方式** | 笔试+面试 |
| **上课地点** | 上海大学悉尼工商学院 |
| **批准书编号** | MOE31AUA02DNR19940091O |

## 项目特色

- **AACSB认证**：上海大学悉尼工商学院2020年通过AACSB认证
- **QS世界前100**：悉尼科技大学2024年QS世界排名第90位
- **1.5年短学制**：快速获得国际认可的硕士学位
- **中国商学院TOP10**：2021年获中国商学院最佳中外合作项目TOP10
- **2024年QS金融硕士全球第70**：金融硕士项目获国际权威排名认可

## 招生条件

### 第一类
- 拥有本科毕业证及学位证

### 第二类
- 本科单证或大专文凭
- 至少15年相关行业中高层管理经验
- 需提交个人简历及工作证明

### 第三类
- 大专或本科单证
- MEM专业：至少2年相关行业工作经验
- MF专业：至少4年相关行业工作经验

## 考核方式

| 项目 | MEM | MF |
|-----|-----|-----|
| 笔试科目 | 综合能力测试（90分钟） | 综合能力测试（90分钟） |
| 面试形式 | 导师组面试（15-20分钟） | 导师组面试（15-20分钟） |
| 总成绩 | 笔试50%+面试50% | 笔试50%+面试50% |

## MEM核心课程

- 工程管理原理
- 项目计划与控制
- 风险管理与质量工程
- 供应链管理
- 技术创新与商业化
- 系统工程与决策分析

## MF核心课程

- 公司金融
- 投资学
- 金融市场与机构
- 衍生品定价
- 风险管理
- 国际金融

## 毕业获证

- 悉尼科技大学工程管理硕士/金融硕士学位证书
- 上海大学悉尼工商学院硕士研究生学业证明
- 可申请教育部留学服务中心学历学位认证
