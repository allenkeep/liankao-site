# 河海大学工程管理硕士（MEM）

## 项目特色

河海大学MEM项目依托学校水利工程学科优势：

- **水利特色鲜明**：依托学校在水利、水电、水运等领域的传统优势
- **工程领域广泛**：覆盖水利工程、土木工程、交通工程等领域
- **复合型培养**：工程技术与管理知识融合
- **实践导向**：紧密结合重大水利工程培养人才

## 研究方向

| 方向 | 简介 |
|------|------|
| 水利工程管理 | 水利工程项目全生命周期管理 |
| 土木工程管理 | 土木工程项目管理 |
| 工程项目管理 | 一般工程项目管理 |

## 报考条件

- 大学本科毕业后有3年以上工作经验
- 高职高专毕业或本科结业后有5年以上工作经验
- 获硕士、博士学位后有2年以上工作经验

**注**：仅招收在职定向就业人员

## 考试科目

- 初试：199管理类综合能力 + 204英语二
- 复试：专业课笔试、思想政治理论，综合面试

## 证书授予

完成规定学分、通过学位论文答辩后，授予：
- 河海大学硕士研究生毕业证书
- 工程管理硕士学位证书

**来源**：以学校官方公布信息为准
