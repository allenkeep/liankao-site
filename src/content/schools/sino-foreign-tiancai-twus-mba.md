# 天津财经大学-加拿大西三一大学工商管理硕士

## 项目基本信息

| 项目名称 | 内容 |
|---------|------|
| **中方院校** | 天津财经大学 |
| **外方院校** | 加拿大西三一大学 (Trinity Western University) |
| **专业方向** | 工商管理 (MBA) |
| **学制** | 14个月 |
| **总学费** | 约16万元（另加3.5万异地教学费） |
| **是否留服认证** | 是 |
| **上课方式** | 周末/集中授课 |
| **招生人数** | 80人/年 |
| **入学方式** | 自主招生 |
| **上课地点** | 北京/上海教学点 |
| **批准书编号** | MOE12CA1A19960319O |

## 项目特色

- **历史悠久**：1987年开始合作办学，是国内最早的中外合作MBA项目之一
- **国际认证**：西三一大学获加拿大政府认证，学位受国际认可
- **学分制无论文**：无需撰写毕业论文，修满学分即可毕业
- **小班授课**：严格控制班级规模，确保教学质量
- **中英双语**：中英双语授课，语言要求相对友好

## 招生条件

1. 本科毕业并获得学士学位
2. 三年以上工作经验
3. 一定的英语基础
4. 通过项目入学考核

## 核心课程

- 管理学原理
- 组织行为学
- 市场营销
- 公司财务
- 战略管理
- 人力资源管理
- 国际商务
- 管理信息系统

## 毕业获证

- 加拿大西三一大学工商管理硕士（MBA）学位证书
- 可申请中国教育部留学服务中心学历学位认证
