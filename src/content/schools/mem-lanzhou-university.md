# 兰州大学工程管理硕士（MEM）

## 基本信息

| 项目 | 信息 |
|------|------|
| **院校名称** | 兰州大学 |
| **项目名称** | 工程管理硕士（MEM） |
| **培养学院** | 管理学院 |
| **项目类型** | 非全日制 |
| **学制** | 3年 |
| **学费** | 以学校最终公布为准 |
| **上课方式** | 周末授课（每月2个周末） |

## 项目特色

兰州大学MEM项目依托学校综合性大学优势：

- **综合性大学优势**：依托学校在化学、大气科学、生态学等领域的深厚积累
- **西部特色鲜明**：深度融合西部地区基础设施建设、能源化工、生态环境等需求
- **双导师制**：校内学术导师+行业资深专家联合指导
- **实践导向**：与大型工程企业合作，强化解决实际问题的能力
- **高性价比**：学费在同类"双一流"A类高校中具有竞争力

## 课程设置

### 核心课程
- 工程管理导论
- 系统工程与运筹学
- 工程经济学
- 工程项目管理
- 工程风险管理
- 质量管理

### 特色方向课
根据西部地区发展需求设置相关选修课程

## 报考条件

- 大学本科毕业后有3年以上工作经验
- 高职高专毕业或本科结业后有5年以上工作经验
- 获硕士、博士学位后有2年以上工作经验

**注**：仅招收在职定向就业人员

## 考试科目

- 初试：199管理类综合能力 + 204英语二
- 复试：思想政治理论笔试、专业综合面试

## 证书授予

完成规定学分、通过学位论文答辩后，授予：
- 兰州大学硕士研究生毕业证书（注明"非全日制"）
- 工程管理硕士学位证书
