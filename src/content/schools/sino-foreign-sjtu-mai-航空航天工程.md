## 项目简介
上海交通大学与俄罗斯莫斯科航空学院合作举办航空航天工程，经教育部批准设立（批准书编号：MOE31RU1A20192029N）。项目免全国联考，采用申请制入学，毕业获外方硕士学位，可在中国教育部留学服务中心进行学历学位认证。

## 审批信息

| 项目信息 | 详情 |
|---------|------|
| 中方办学者 | 上海交通大学 |
| 外方办学者 | 俄罗斯莫斯科航空学院 |
| 开设专业 | 航空航天工程 |
| 批准书编号 | MOE31RU1A20192029N |
| 学制 | 2.5年 |
| 每期招生人数 | 40人 |
| 招生起止年份 | 2020年—2026年 |
| 招生方式 | 纳入国家硕士研究生招生计划 |
| 学费 | 约77万元（141,800新币） |
| 学历认证 | 可留服认证 |

## 申请条件
- 本科及以上学历
- 相关专业背景优先
- 具备一定英语能力
- 具体条件以当年招生简章为准

## 项目特色
- 教育部批准，学历可留服认证
- 免全国联考，申请制入学
- 中外双方师资联合授课
- 国际化课程体系
