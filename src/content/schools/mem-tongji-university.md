# 同济大学工程管理硕士（MEM）

## 基本信息

| 项目 | 信息 |
|------|------|
| **院校名称** | 同济大学 |
| **项目名称** | 工程管理硕士（MEM） |
| **培养学院** | 经济与管理学院 |
| **项目类型** | 非全日制 |
| **学制** | 2.5-3年 |
| **学费** | 26.8万元（全程） |
| **上课方式** | 周末授课+集中授课 |
| **授课地点** | 同济大学四平路校区（上海） |
| **招生人数** | 参考往年计划 |
| **复试线** | 参考同济大学自主划线 |
| **院校属性** | 985、211、双一流A类 |
| **官方网址** | https://www.tongji.edu.cn |

## 项目方向

- 工程项目管理
- 房地产开发与管理
- 项目计划与控制
- 工程建设与咨询
- 建设工程法律与合同管理

## 项目特色

- **土木工程全球第一**：在土建、交通领域享有盛誉
- **德式精英教育**：与德国高校深度合作，国际化特色鲜明
- **国家超级工程参与**：港珠澳大桥、北京大兴机场、上海中心大厦等
- **国际认证**：经济与管理学院通过AACSB、EQUIS、AMBA三大认证
- **行业地位**：被誉为中国基建的"黄埔军校"
- **IPMP认证合作**：与IPMP国际项目经理资质认证深度合作

## 课程设置

### 核心课程
- 工程管理导论
- 系统工程
- 工程经济学
- 项目管理
- 质量管理
- 风险管理
- 工程建设合同管理
- BIM原理与应用

### 方向选修课
- 房地产开发与经营
- 工程造价管理
- 城市管理
- 智慧城市

## 报考条件

1. 大学本科毕业后有3年以上工作经验
2. 获得国家承认的高职高专毕业学历或大学本科结业后，有5年以上工作经验
3. 获得硕士、博士学位后有2年以上工作经验

## 初试科目

| 科目 | 满分 |
|------|------|
| 管理类联考综合能力（199） | 200分 |
| 英语二（204） | 100分 |

## 证书授予

完成培养方案规定学分、通过学位论文答辩，经同济大学学位评定委员会审核批准，可获得：
- 同济大学硕士研究生毕业证书（非全日制）
- 工程管理硕士（MEM）学位证书
