# 华南理工大学MEM项目介绍

## 学校简介

华南理工大学是教育部直属的全国重点大学，国家"双一流""211工程""985工程"重点建设高校。学校位于广州，拥有悠久的办学历史和深厚的学术底蕴。

## MEM项目简介

华南理工大学工程管理硕士（MEM）项目由土木与交通工程学院承办，依托学校在土木工程、建筑学等领域的优势学科，培养高层次工程管理人才。

### 研究方向

- 工程项目管理
- 土木工程建造与管理
- 房地产开发与管理
- 工程经济与造价管理

### 招生信息

| 项目 | 内容 |
|------|------|
| 招生院系 | 土木与交通工程学院 |
| 学习方式 | 非全日制 |
| 授课方式 | 周末授课 |
| 招生人数 | 约80人 |

### 学制学费

| 项目 | 内容 |
|------|------|
| 学制 | 3年 |
| 学费 | 15万元（全程，5万/年） |
| 授课地点 | 广东广州 |

## 报考条件

- 本科毕业后有3年以上工作经验
- 高职高专毕业后有5年以上工作经验
- 硕士/博士学位后有2年以上工作经验

## 师资力量

学院在土木工程领域拥有一批知名专家学者，承担了多项国家重大工程项目。

## 就业前景

毕业生主要就业于大型建筑企业、房地产公司、设计院、跨国公司等，就业前景广阔。
