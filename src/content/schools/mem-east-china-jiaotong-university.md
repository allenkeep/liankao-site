# 华东交通大学MEM（工程管理硕士）

## 学校简介

华东交通大学（East China Jiaotong University）坐落于江西省南昌市，是一所以交通为特色、轨道为核心、多学科协调发展的教学研究型大学。学校原隶属铁道部，2000年转制为"中央与地方共建，以地方管理为主"，是国铁集团与江西省人民政府"双共建"高校、"中西部高校基础能力建设工程"高校、博士学位授权单位。

学校在交通运输工程、土木工程、电气工程等领域具有鲜明特色。

## 招生学院

**土木建筑学院**

## 项目信息

| 项目 | 内容 |
|------|------|
| 学习方式 | 全日制、非全日制 |
| 学制 | 3年 |
| 全日制学费 | 3.6万/全程 |
| 非全日制学费 | 4.8万/全程 |

## 研究方向

- 土木工程建设管理
- 资源与环境工程建设管理

## 培养特色

1. **交通行业优势**：依托深厚的交通行业底蕴，与中国铁路南昌局集团等企业共建实践基地

2. **就业前景广阔**：毕业生就业去向多元化，多进入铁路、基建、电力等行业

## 报考条件

1. 大学本科毕业后有3年及以上工作经验；
2. 获得国家承认的高职高专毕业学历后，有5年及以上工作经验；
3. 获得硕士、博士学位后有2年及以上工作经验。
