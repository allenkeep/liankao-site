## 项目简介

天津财经大学与加拿大西三一大学(Trinity Western University)合作的MBA项目，是教育部批准的中外合作办学项目。2年学制，毕业可留服认证。

## 审批信息

| 项目 | 详情 |
|-----|------|
| 办学地址 | 天津市河西区珠江道25号 |
| 中外方办学者 | 中方：天津财经大学；外方：加拿大西三一大学 |
| 办学层次 | 外国硕士学位教育 |
| 学制 | 2年 |
| 每期招生人数 | 90人 |
| 招生起止年份 | 1995年—2027年（每年1期） |
| 招生方式 | 自主招生（招生标准不低于加拿大西三一大学在加拿大的标准） |
| 开设专业 | 工商管理 |
| 颁发证书 | 中方：无；外方：MBA工商管理硕士学位证书 |
| 审批机关 | 教育部 |
| 批准书编号 | MOE12CA1A19950194O |
| 批准书有效期 | 2029年12月31日 |

> 信息来源：教育部中外合作办学监管工作信息平台
