# 华东理工大学-澳大利亚堪培拉大学工商管理硕士

## 项目基本信息

| 项目名称 | 内容 |
|---------|------|
| **中方院校** | 华东理工大学 |
| **外方院校** | 澳大利亚堪培拉大学 (University of Canberra) |
| **专业方向** | 工商管理 (MBA) |
| **学制** | 2年（在职学习，最长4年） |
| **总学费** | 27.8万元 |
| **是否留服认证** | 是 |
| **上课方式** | 周末隔周上课，面授 |
| **招生人数** | 140人/年 |
| **入学方式** | 自主招生（综合能力面试） |
| **上课地点** | 华东理工大学徐汇校区 |
| **批准书编号** | MOE31AU1A20000511O |

## 项目特色

- **历史悠久**：2000年获教育部批准，办学26年，培育1600余名校友
- **五项国际认证**：AACSB、EQUIS、AMBA、BGA、CAMEA
- **无需统考**：自主招生，综合能力面试录取
- **英文授课**：专业术语与实践能力并重
- **无需出国**：全程在华东理工大学完成学习
- **国际视野**：澳大利亚学历资格框架(AQF)体系

## 招生条件

学术背景要求（满足其一）：
- 相关领域学士学位，无工作经验要求
- 非相关学士学位，至少3年相关全职工作经验
- 相关研究生学位或研究生文凭，无工作经验要求
- 高级文凭+4年相关工作经验
- 文凭+8年工作经验

英语要求（满足其一）：
- 雅思(学术类) 总分6.5分，单项不低于6.0分
- 托福iBT 总分79分
- 培生英语(学术类) 总分58分
- 或参加项目自主英语能力考察

## 核心课程

- 经济与决策科学
- 管理与市场营销
- 财务与金融投资
- 创业与战略创新
- 领导力培养
- 商业分析

## 毕业获证

- 堪培拉大学工商管理硕士（MBA）学位证书
- 教育部留学服务中心国外学历学位认证书
- 华东理工大学结业证书
